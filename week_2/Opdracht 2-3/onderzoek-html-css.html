4. Waarom ziet een website er soms net iets anders uit in verschillende browsers, zoals Chrome, Firefox, Edge of Safari?

Omdat elke browser zijn eigen standaard instellingen heeft, zien de zelfde webpagina's er een beetje anders uit op verschillende browsers.
Iedere web browser heeft zijn eigen manier van waarop hij de code begrijpt/uitvoert. Soms ligt het ook aan de CSS script die misschien voor een specifieke browser geschreven is
